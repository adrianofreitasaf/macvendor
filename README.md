# MACVendor

### A simple script in php to get the MAC address of the manufacturer entered

###Dependencies: 
- sudo apt-get install curl php5 php5-curl

### HOW TO INSTALL
- chmod +x install.sh
- sudo ./install.sh

### HOW TO USE:
- macvendor <MAC_ADDRESS>

### EXAMPLE
- macvendor AA:BB:CC:DD:EE:FF
